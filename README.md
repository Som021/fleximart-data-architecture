# FlexiMart Data Engineering Project

## Project Overview
This repository contains a complete end-to-end data engineering solution for the FlexiMart retail system.  
The project demonstrates practical implementation of:

- Relational Database Design & ETL
- Data Quality Management
- NoSQL (MongoDB) Modeling
- Data Warehouse (Star Schema) & Analytics
- Business Intelligence Queries

The project is divided into three structured parts, each focusing on a different data architecture concept.
